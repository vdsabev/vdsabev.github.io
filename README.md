# vdsabev's blog

Personal blog powered by **[cmless](https://github.com/vdsabev/cmless)**.

- Posts are written and published as GitHub Issues (apply `status: published` or `status: unlisted` labels).
- Full Markdown support + drag-and-drop images (hosted by GitHub).
- No database, no content commits — the site rebuilds automatically on label/issue changes.

See the [cmless documentation](https://github.com/vdsabev/cmless) for how it works, writing posts, local development, configuration, and keeping your blog in sync with upstream improvements.
